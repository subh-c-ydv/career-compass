# The Career Compass Map — Framework Reference

## Overview

The Career Compass Map is a career coaching framework that structures professional identity across multiple independently defined dimensions, held together by a central hub — the personal brand. Unlike fixed frameworks that prescribe what the dimensions should be, the Career Compass Map is built around the individual: the coach and client define the dimensions together.

## Core Elements

### Dimensions (3–6)

Each dimension represents a distinct aspect of the person's professional identity. A dimension might be:

- A functional expertise (e.g. Software Delivery Leadership)
- An aspirational direction (e.g. Strategy & Consulting)
- A methodological identity (e.g. Agile Coaching)
- A sector or industry focus
- A leadership or people capability
- A creative or entrepreneurial pursuit

The discipline of choosing and naming dimensions is itself a coaching exercise — it forces clarity about what actually defines the career.

### The Hub

The hub is the personal brand: the through-line narrative that makes sense of all dimensions simultaneously. A weak hub means the dimensions feel like separate careers. A strong hub means they feel like facets of one coherent professional.

### Coherence

The framework is most powerful when the dimensions are:
1. **Distinct** — not overlapping or redundant
2. **Connected** — each one reinforces the others
3. **Coherent with the hub** — all flow from the same core identity

## The Radar Chart

The radar chart is the primary diagnostic visual. Each axis represents one dimension, scored 1–10. The shape of the filled polygon tells the story:

| Shape | Interpretation |
|---|---|
| Balanced polygon | Strong coherence — dimensions reinforce each other |
| Lopsided shape | Imbalance — one or more dimensions underdeveloped |
| Small overall | Early stage — career still building depth across dimensions |
| Large but thin | Breadth without depth in some areas |

A lopsided shape is not a failure — it is a coaching opportunity. The gap between the current shape and the ideal shape is where the work happens.

## Scoring (1–10)

| Score | What it means |
|---|---|
| Coherence Score | Overall cross-dimension narrative fit |
| Hub Strength | How clear and compelling the personal brand is |
| Dimension Scores | Depth, credibility, and development in each individual dimension |

## How Career Compass uses it

1. Coach defines 3–6 dimensions for the person being profiled
2. Each dimension is described with a label, description, years of experience, and key highlights
3. The hub statement and core values are captured
4. Claude analyses coherence, scores each element, and surfaces tensions and opportunities
5. The radar chart renders the shape of the career visually
6. Coaching questions are generated to guide the session
